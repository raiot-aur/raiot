Smart WiFi

Objective

The objective of the project is to have a better downloading system integrated into a personal network and is also capable of cloud storage.

Methodology & Brief

The S_P_N or smart personal network is network based storage and downloader solution for personal use it consist of storage device, a rasberry pi and a router.The end user interacts with the web application which is acessed through to domain name of the pi, because the pi is connected to the network(internet) ,it make it enable even outside the that particual network hard connected with pie.The user can choose weather he wants to download to the storage or want acess a particular file from the storage.The downloads can be seed remotely and the priority of network usage can also be controlled making it much more fessible to use.

Features

  • Cost effective
  • Ready to use out of the box
  • Safe and Secure
